# Sistema de Gestión de Citas Médicas

Este repositorio contiene el diseño y la estructura base del sistema de gestión para la clínica médica, desarrollado como parte de la actividad de **Entornos de Desarrollo**.

## Estructura del Proyecto

* **`/analisis`**: Contiene la documentación inicial y diagramas de requisitos (Casos de Uso).
* **`/diseno`**: Contiene el **Diagrama de Clases** detallado (Modelo de Dominio) exportado desde Visual Paradigm. (La herramienta CASE que elegimos). Además de dos imagenes, una es una captura de pantalla porque la **imagen del diagrama exportada** por VP, salío con marca de agua dificultando la visibilidad.
* **`/src`**: Contiene el **código fuente Java** generado automáticamente mediante Visual Paradigm. Incluye la jerarquía de usuarios, citas médicas y gestión de informes.
* **`/docs`**: Documentación adicional del proyecto.

## Herramientas Utilizadas

* **Visual Paradigm**: Para el modelado UML y la ingeniería de software (CASE).
* **Java**: Lenguaje de programación base para la estructura del sistema.
* **Git/GitHub**: Para el control de versiones y trabajo colaborativo.
